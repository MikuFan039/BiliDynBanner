## `header`文件夹

这个文件夹是哔哩哔哩部分特殊动态 banner 的元数据

关键部分：`extensions`字段下的内容

示例内容：

```json
\"extensions\":{\"springGame2022\":{}}
```

---

### 各种扩展对应的元数据：

1. `time`扩展：`time`文件夹

2. 高级/游戏扩展：`advExt`文件夹

- `springGame2022`扩展：`2022spring.json`
- `summer2022`扩展：`2022summer.json`
- `autumn2022`扩展：`2022autumn.json`

### 旧版分层数据：

`2020autumn.json`，对应`frontend/2020autumn.js`
