## `animated-banner` 动态 Banner 组件

**此处是动态 Banner 的核心组件，保持官方实现**

对`2020autumn`动态 Banner 做了兼容处理，修复了不播放眨眼动画的 Bug

---

来源：使用`reverse-sourcemap`还原的`header.js`源码

文件 url：

```
http://s1.hdslb.com/bfs/seed/jinkela/header-v2/header.js
http://s1.hdslb.com/bfs/seed/jinkela/header-v2/header.js.map
```

---

## 食用方式

### 1. 安装`reverse-sourcemap`

```bash
npm install -g reverse-sourcemap
```

### 2. 还原`*.map`文件

```bash
reverse-sourcemap header.js.map
```
